# Metaheuristics for the Traveling Salesperson Problem

This project studies the Traveling Salesperson Problem (TSP) and applies two metaheuristic algorithms:
- Ant Colony Optimization (ACO)
- Simulated Annealing (SA)

## Files
- `aco.py`: Ant Colony Optimization implementation
- `sa.py`: Simulated Annealing implementation
- `tsp_utils.py`: Utility functions and data generation
- `experiments.py`: Runs experiments and prints results

## How to Run
Make sure Python 3 is installed. Then run:

```bash
python experiments.py
```
